# 派对

## 题目描述

有 $n$ 个人参加聚会，在现场没有朋友的人会先率先离开，然后依次是在现场有 1 个朋友，有 2 个朋友，有 3 个朋友... 的人依次离开，求最后最多剩下多少人。

## 输入描述

第一行仅有一个整数 $t$ ($1 \le t \le 10^5$) 表示测试数据的组数；接下来的 $t$ 行，每行给出一组测试数据，每组测试数据仅有一个整数 $n$ ($1 \le n \le 10^5$) 表示初始人数。

## 输出描述

每组样例输出最多能剩下的人数。

## 样例

**输入**
```
1
3
```

**输出**
```
1
```
