# 坐标转换

## 题目描述

在电子表格软件中，有两种坐标表示方法。

**第一种**：列号用大写字母表示，第 1 列被标为 `A`，第 2 列为 `B`，...，第 26 列为 `Z`，第 27 列为 `AA`，第 28 列为 `AB`...。`ZZ` 列之后则由三个字母构成列号 `AAA`、`AAB`...，如此类推。行号用从 1 开始的数字进行表示。比如，`BC23` 表示位于第 55 列 23 行的单元格。

**第二种**：直接用行号和列号表示坐标的方法：`RXCY`，其中 X 和 Y 分别表示行号和列号。比如 `BC23` 用这种表示方法表示即为 `R23C55`。

你的任务是完成这两种坐标表示方法的相互转换。

## 输入描述

第一行仅有一个整数 $n$ ($1 \le n \le 10^5$)，表示需要转换的坐标个数。

接下来的 $n$ 行每行给出一个单元格的坐标，数据保证输入格式正确且行和列的数量不会超过 $10^6$。

## 输出描述

共有 $n$ 行，依次输出坐标转换的答案。

## 样例

**输入**
```
2
R23C55
BC23
```

**输出**
```
BC23
R23C55
```
