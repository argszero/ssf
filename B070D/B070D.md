
## 题目描述

给定一棵有 $N$ 个顶点的树。

树是一种图，当顶点数为 $N$ 时，它是一个没有闭环的连通图，边的数量为 $N-1$。

第 $i$ $(1 \le i \le N-1)$ 条边连接顶点 $a_i$ 和顶点 $b_i$，距离为 $c_i$。

另外，给定 $Q$ 个查询和整数 $K$。

- 在第 $j$ $(1 \le j \le Q)$ 个查询中，请计算从顶点 $x_j$ 经过顶点 $K$ 到达顶点 $y_j$ 的最短路径距离。

## 输入描述

第一行给出了树的顶点数量 $N$，接下来 $N-1$ 行，每行包含一条边，$a_i$、$b_i$ 和 $c_i$；

紧接着给出了查询数量 $Q$ 和整数 $K$，它们以空格分隔。

接下来的 $Q$ 行给出了每个查询的顶点 $x_j$ 和 $y_j$。

## 输出描述

对于每次循环输出一个值表示最短距离。

## 样例输入 1

```
5
1 2 1
1 3 1
2 4 1
3 5 1
3 1
2 4
2 3
4 5
```

## 样例输出 1

```
3
2
4
```

## 样例输入 2

```
7
1 2 1
1 3 3
1 4 5
1 5 7
1 6 9
1 7 11
3 2
1 3
4 5
6 7
```

## 样例输出 2

```
5
14
22
```

## 样例输入 3

```
10
1 2 1000000000
2 3 1000000000
3 4 1000000000
4 5 1000000000
5 6 1000000000
6 7 1000000000
7 8 1000000000
8 9 1000000000
9 10 1000000000
1 1
9 10
```

## 样例输出 3

```
17000000000
```

## 提示

### 数据范围

- $3 \le N \le 10^5$
- $1 \le a_i, b_i \le N$ $(1 \le i \le N-1)$
- $1 \le c_i \le 10^9$ $(1 \le i \le N-1)$
- $1 \le Q \le 10^5$
- $1 \le K \le N$
- $1 \le x_j, y_j \le N$ $(1 \le j \le Q)$
- $x_j \ne y_j$ $(1 \le j \le Q)$
- $x_j \ne K, y_j \ne K$ $(1 \le j \le Q)$
